# Technical Authority

## Purpose
Define roles responsible for technical decisions and system integrity.

## Roles

- System Engineer → overall technical coherence
- Lead Architect → architecture decisions
- Verification Lead → verification integrity
- Project Manager → program alignment

## Responsibilities

- approve technical baselines
- resolve technical conflicts
- ensure compliance with requirements

## Engineering Implications

- clear responsibility
- reduced ambiguity
- stronger governance

# Review Gates and Milestones

## Purpose
Define key decision points in the system lifecycle.

## Main Review Gates

### SRR – System Requirements Review
- Validate requirements completeness
- Confirm stakeholder alignment

### PDR – Preliminary Design Review
- Assess architecture maturity
- Confirm design feasibility

### CDR – Critical Design Review
- Validate detailed design
- Confirm readiness for implementation

## Milestone Logic

Each gate must:
- have entry criteria
- have exit criteria
- produce review outputs

## Engineering Implications

- structured lifecycle progression
- risk reduction
- controlled decision making
